# **App Name**: FreeZone Store

## Core Features:

- Custom Sign-Up/Login: Create and log in users with email and password, with bcrypt password hashing.
- Play Store UI Design: Bottom navigation for Home, Apps, Search, Books mirroring Google Play Store's UI.
- Voice Search (AI-Like): Capture user input via microphone and convert it to text using native Speech-to-Text.
- Search & Categories: Trending searches, chip-based categories, and full-text search capabilities.
- App Listing Display: Display app icon, name, developer, rating, and description.
- Download/Installation: Button to download the app's APK file from your own Node.js server. (Download link should be secure).
- Rating & Reviews: Allow logged-in users to give ratings (1-5 stars) and post reviews for apps.

## Style Guidelines:

- Primary color: Deep teal (#008080) to evoke a sense of technology and trust.
- Background color: Light gray (#F0F0F0) to ensure readability and a modern feel.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights.
- Headline font: 'Poppins' (sans-serif) for headlines to convey a precise, contemporary feel.
- Body font: 'PT Sans' (sans-serif) for body text, for a modern look with warmth and personality. Used in conjunction with 'Poppins'.
- Simple, consistent icons to represent different app categories and actions.
- Grid-based layout for app listings and clear separation of content sections.
- Subtle animations for transitions and feedback on user interactions.
